# School Management React Application

A modern icon-rich School Management Dashboard built with React JS.

## 🚀 Features
- **Full CRUD**: Add, Edit, Delete, and List schools.
- **Search**: Interactive search bar for filtering schools.
- **Animations**: Smooth transitions using Framer Motion.
- **Design**: Premium glassmorphism UI.

## 📦 Run it locally
1. `npm install`
2. `npm run dev`
